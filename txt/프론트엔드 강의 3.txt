<!DOCTYPE html> <- 문서의 html 버전을 지정

html - 문서의 전체 범위

head - 문서의 정보를 나타내는 범위

body - 문서의 구조를 나타내는 범위

head 부분
title - html 제목
link = 외부 문서를 가져와 연결할 때 사용 rel="가져올 문서와의 관계" href="가져올 문서의 경로"  
icon - 제목 옆 로고 파피콘 이미지
meta = 정보 name = 정보의 종류 content = 정보의 값
charset = 문자인코딩 방식

상대경로 vs 절대경로
./ (생략가능)        http(https)
../                      /(//)
